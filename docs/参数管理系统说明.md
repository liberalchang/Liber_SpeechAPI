# 参数管理系统说明文档

## 概述

Liber Speech API 现在支持全局默认参数管理系统，允许用户设置全局默认值，简化日常使用。系统采用分层参数管理：

1. **全局默认参数** - 存储在 `src/liber_speech/config/default_params.json`
2. **用户自定义参数** - 存储在 `src/liber_speech/config/user_{module}_params.json`
3. **命令行参数** - 运行时传入的参数，优先级最高

## 参数优先级

```
命令行参数 > 用户自定义参数 > 全局默认参数 > 环境配置(.env)
```

## 使用方法

### 1. 修改全局默认参数

直接编辑 `src/liber_speech/config/default_params.json` 文件：

```json
{
  "tts": {
    "model": "turbo",
    "language": "zh",
    "format": "wav"
  }
}
```

### 2. 命令行使用示例

#### ASR 命令

```bash
# 使用全局默认参数
.venv\Scripts\python.exe -S scripts\run_cli_fixed.py asr transcribe "audio.wav"

# 覆盖部分参数
.venv\Scripts\python.exe -S scripts\run_cli_fixed.py asr transcribe "audio.wav" --model "whisper-medium" --language "en"

# 完全自定义参数
.venv\Scripts\python.exe -S scripts\run_cli_fixed.py asr transcribe "audio.wav" --model "whisper-small" --language "zh" --timestamps "word" --flash
```

#### TTS 命令

```bash
# 使用全局默认参数
.venv\Scripts\python.exe -S scripts\run_cli_fixed.py tts synthesize "你好世界"

# 覆盖部分参数
.venv\Scripts\python.exe -S scripts\run_cli_fixed.py tts synthesize "你好世界" --model "turbo" --format "mp3"

# 完全自定义参数
.venv\Scripts\python.exe -S scripts\run_cli_fixed.py tts synthesize "Hello World" --model "standard" --language "en" --output "hello.wav" --temperature 0.9
```

## 参数说明

### ASR 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| model | string | "openai/whisper-large-v3" | Whisper 模型名称 |
| device | string | "auto" | 推理设备 (auto/cpu/cuda:N) |
| language | string | null | 语言代码，null 为自动检测 |
| task | string | "transcribe" | 任务类型 (transcribe/translate) |
| timestamp | string | "chunk" | 时间戳级别 (chunk/word) |
| batch_size | int | 24 | 批次大小 |
| flash | bool | false | 是否启用 Flash Attention 2 |
| hf_token | string | null | Hugging Face 访问令牌 |
| diarization_model | string | "pyannote/speaker-diarization-3.1" | 说话人分割模型 |
| num_speakers | int | null | 说话人数量 |
| min_speakers | int | null | 最小说话人数量 |
| max_speakers | int | null | 最大说话人数量 |
| transcript_path | string | "output.json" | 输出文件路径 |

### TTS 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| model | string | "multilingual" | TTS 模型 (turbo/standard/multilingual) |
| device | string | "auto" | 推理设备 (auto/cpu/cuda:N) |
| language | string | "zh" | 语言代码 |
| format | string | "wav" | 输出格式 (wav/mp4/ogg_opus) |
| repetition_penalty | float | 1.2 | 重复惩罚系数 |
| temperature | float | 0.8 | 采样温度 |
| top_p | float | 0.95 | 核采样概率阈值 |
| top_k | int | 1000 | 最高 K 个候选 token 采样 |
| norm_loudness | bool | true | 是否标准化音频响度 |
| exaggeration | float | 0.0 | 夸张度系数 |
| cfg_weight | float | 0.0 | 无分类器引导权重 |
| audio_prompt_path | string | null | 参考音频路径 |
| output_path | string | "output.wav" | 输出文件路径 |

## 输出路径处理

### ASR 输出路径

- 如果指定了 `LIBER_RESULTS_DIR` 环境变量，相对路径会基于该目录
- 否则相对路径基于当前工作目录
- 绝对路径直接使用

### TTS 输出路径

- 如果指定了 `LIBER_RESULTS_DIR` 环境变量，相对路径会基于该目录
- 否则相对路径基于当前工作目录
- 绝对路径直接使用

## 编程接口

### 参数管理器使用

```python
from liber_speech.config.params_manager import create_asr_manager, create_tts_manager

# 创建 ASR 参数管理器
asr_manager = create_asr_manager()
user_params = {"model": "whisper-medium", "language": "zh"}
params = asr_manager.get_all_params(user_params)

# 创建 TTS 参数管理器
tts_manager = create_tts_manager()
user_params = {"model": "turbo", "temperature": 0.9}
params = tts_manager.get_all_params(user_params)
```

### 保存用户参数

```python
# 保存用户自定义参数
asr_manager.save_user_params({"model": "whisper-small", "batch_size": 16})

# 加载用户参数
saved_params = asr_manager.load_user_params()

# 重置为默认参数
asr_manager.reset_to_defaults()
```

## 故障排除

### 1. 参数不生效

- 检查 `default_params.json` 格式是否正确
- 确认参数名称拼写正确
- 查看命令行是否有语法错误

### 2. 模型加载失败

- 检查模型名称是否正确
- 确认网络连接正常
- 查看缓存目录权限

### 3. 输出路径问题

- 确认目录存在且有写权限
- 检查 `LIBER_RESULTS_DIR` 环境变量设置
- 使用绝对路径测试
